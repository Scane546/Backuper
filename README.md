# Backuper
Программа на скриптах для автоматического создания бэкапов во время игры или работы. Работает во время того как приложение запущено и создает бээкапы по промежутку времени указонному вами месте/A script program for automatically creating backups while playing or working. It works while the application is running and creates backups based on the time interval specified by you.
